# What is SAP?

**SAP** is a German software company that builds software for businesses.

Think of SAP like a **central brain/system for a company**.

Instead of using:
- One app for finance  
- Another for inventory  
- Another for HR  
- Another for sales  

👉 SAP combines everything into **one connected system**.

---

## Example

A company sells a laptop.  

SAP can automatically:
- Reduce stock count  
- Generate invoice  
- Update accounting  
- Notify warehouse  
- Track delivery  
- Show profit/loss  

✅ All in one system
